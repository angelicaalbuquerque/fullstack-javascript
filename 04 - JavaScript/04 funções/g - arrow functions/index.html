As arrow functions também são funções anônimas, mas com uma sintaxe reduzida. Sendo assim, podemos entendê-las como uma simplificação da função anônima. 

          const doubleVelocity = (velocity) => {
            return velocity * 2;
          };
          
          console.log(doubleVelocity(60));

Quando só tem uma linha, podemos simplificar como:

          const doubleVelocity = (velocity) => velocity * 2;

          console.log(doubleVelocity(60));

Quando só tem um parâmetro, podemos omitir os parênteses:

          const doubleVelocity = velocity => velocity * 2;

          console.log(doubleVelocity(60));
