Funções que têm nome omitido no momento da criação. Utilizada, geralmente, para ser armazenada em variáveis. 

          let doubleSpeed = function(velocity){
            return velocity * 2
          }

          let newVelocity = doubleSpeed(40)
          console.log(newVelocity)

A declaração de uma função na forma tradicional faz com q ela seja lida antes de tudo no escopo do código, como acontece no caso abaixo.

Porém, as funções anônimas, por serem atribuídas em variável, seguem a mesma regra de chamada de variáveis e o que procava erro no caso abaixo, já que foi declarada com let e não foi inicializada. 

          console.log(speedUp(60, 10));
          console.log(doubleSpeed(50));

          let doubleSpeed = function (velocity) {
            return velocity * 2;
          };

          function speedUp(velocity, acceleration) {
            return velocity + acceleration;
          }
