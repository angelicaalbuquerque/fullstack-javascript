Parâmetros

Funções podem receber dados quando são chamados. E isso ocorre através dos parâmetros, que são bem úteis quando queremos que as funções executem uma tarefa, mas utilize algum dado que estamos enviando.

Esses parâmetros passam a ser variáveis dentro das funções.

Parâmetros com valo padrão

Quando criamos função, podemos colocar valores padrão, ou seja, são valores que o parâmetro vai receber caso a gente não queira passar esse parâmetro na hora da chamada da função.

Armadilhas de Parâmetros

  1. Quando chamamos uma função sem parâmetro necessário: no JS, nenhum parâmetro é, de fato, obrigatório. Como um parâmetro é como uma variável dentro da função, por padrão tem valor undefined. E quando não existe nenhum valor padrão definido para o parâmetro, ele imprime undefined.

  2. Ordem dos parâmetros:
    3. function spaceshipSpeedUp(spaceshipVelocity, unit = "km/s",
        spaceshipAcceleration) { 
          let newSpaceshipVelocity = spaceshipVelocity + 
          spaceshipAcceleration; 
          console.log("Nova velocidade: " + newSpaceshipVelocity); 
        } 
        
        spaceshipSpeedUp(50, 20);

    spaceshipVelocityrecebe 50, unit recebe 20 e spaceshipAcceleration recebe undefined. Na hora de fazer a soma, a função tenta somar 50 + undefined. No JS, como não consegue fazer operação aritmética, retorna o valor NaN (Not a Number).

    Já que a atribuição é feita na mesma ordem que a função foi declarada, se a gente tem um valor padrão entre dois sem, o JS não vai jogar os valores pra quem não tem valor padrão, vai continuar atribuindo na mesma ordem que a função foi declarada. A atribuição sempre vai ocorrer na ordem da declaração.

    Para contornar esse problema, nesse caso, poderíamos colocar "unit" como parâmetro final. Assim, seria exibido "70 km/s". 

    function spaceshipSpeedUp(spaceshipVelocity spaceshipAcceleration, unit = "km/s",) { 
          let newSpaceshipVelocity = spaceshipVelocity + 
          spaceshipAcceleration; 
          console.log("Nova velocidade: " + newSpaceshipVelocity); 
        } 
        
        spaceshipSpeedUp(50, 20);
