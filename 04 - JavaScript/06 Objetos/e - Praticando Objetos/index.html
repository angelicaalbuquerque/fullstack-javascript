Aplicativo que fará o mapeamento de velocidade de uma espaçonave. Nele, vamos cadastrar uma espaçonave com seu nome, tipo e a velocidade máxima que esta nave pode atingir para que não entre em combustão.

  1. A nave deve partir da velocidade 0km/s
  2. Menu deve ser exibido para o usuário perguntando se ele quer acelerar ou parar a nave
    • Se ele quiser parar, uma mensagem deve ser exibida com as características da name (nome e tipo) e em que velocidade ele estava
    • Se ele quiser acelerar, deve ser exibido um prompt para que ele informe quanto ele quer acelerar e este valor será incrementado na velocidade
  3. Quando a velocidade for incrementada, deve-se fazer uma verificação para ver se a espaçoname está acima da velocidade máxima dela. Se estiver, exibir mensagem informando a velocidade da name e a velocidade máxima permitida.

Para resolver este problema, é preciso:

  1. Exibir prompt para nome da nave
  2. Exibir prompt para tipo da nave
  3. Exibir prompt para velocidade máx da nave
  4. Exibir prompt para saber se usuário quer acelerar ou parar
    • Se ele quiser parar, exibir mensagem final
    • Se quiser acelerar, prompt para usuário informar o quanto
    • Exibir mensagem de alerta quando a velocidade máx for ultrapassada
  5. Criar objeto para a nave
