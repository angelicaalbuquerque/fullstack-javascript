Imóveis cadastrados

    Exercício

    Criar um sistema de cadastro de imóveis onde, para cadastrar, é necessário passar as seguintes informações: o tipo (se é casa ou apartamento), a áreea e se está alugado.

    Após o cadastro do imóvel, o formulário deve ser limpo para o próximo cadastro.

    Abaixo desse form, deve haver uma lista com os imóveis que já foram cadastrados e esta lista deve ser atualizada assim que o imóvel é criado no formulário.

    Nesta lista, os imóveis alugados devem estar com a marcação "ALUGADO" antes da primeira informação, com fundo vermelho e letras brancas.

    Também deve haver um botão para remoção de um imóvel cadastrado.

    Para esse sistema, será preciso os conhecimentos de:

    • Classes: no mínimo duas classes nessa aplicação - uma para o imóvel e outra para a aplicação.
    • Manipulação de DOM
    • Chamada de eventos a partir do HTML
