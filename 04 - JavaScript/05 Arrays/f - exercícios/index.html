Dado um array bidimensional de naves que estão engatas numa estação espacial: 

          const hitchedSpaceships = [
            ["Fenix", 8, true],
            ["Golias", 10, true],
            ["Helmet", 5, false],
            ["Elemental", 3, true],
            ["Darwin", 15, false]
          ]

Para cada sub-array, a primeira posição é o nome da nave, a segunda a quantidade de tripulantes e a terceira a informação se o processo de engate já foi concluído ou não.

Nesta lista, a plataforma de engate que a nave está parada é o [índice da nave no array + 1]. Por exemplo, a nave Fenix, a plataforma de engate é 0 + 1.

Com base nesse array, queremos:

  1. Filtrar o nome das naves que têm mais de 9 tripulantes;
  2. Informar o número da plataforma em que está a primeira nave que ainda está com engate pendente -- utilize a função findIndex, que é bem parece com o Find, mas retorna o índice de um filtro em vez do elemento do Array;
  3. Destacar o nome de todas as naves colocando-as em caixa alta e exibindo;
  4. Exibir um alerta om todas estas informações.
